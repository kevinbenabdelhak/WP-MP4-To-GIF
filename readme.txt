Convertissez facilement vos vidéos MP4 en GIF animés directement dans le navigateur, et ajoutez-les à votre bibliothèque de médias WordPress en un clic.

== Description ==

**WP MP4 To GIF** permet de convertir des fichiers vidéo MP4 en images GIF animées, localement, sans solliciter votre serveur. L’enregistrement dans la bibliothèque de médias s’effectue en un clic. Le plugin propose aussi la conversion par lot depuis la Médiathèque (vue liste), avec personnalisation de la résolution et du framerate, afin d’adapter la fluidité et le poids de vos GIFs.

Fonctionnalités principales :

- **Conversion rapide MP4 en GIF dans le navigateur** (aucun upload externe)
- **Enregistrement direct du GIF généré dans la bibliothèque WordPress**
- **Conversion par lot de plusieurs vidéos MP4** depuis la Médiathèque
- **Réglage de la résolution et du nombre d’images/seconde** (qualité/poids)
- **Prévisualisation instantanée du GIF**
- **Compatible multisite et multi-utilisateur**
- **Interface simple et intégrée dans le menu médias**

Facilitez la création de GIFs animés à partir de vidéos pour illustrer vos articles, vos réseaux sociaux ou vos tutoriels directement depuis votre admin WordPress.

== Installation ==

1. Téléchargez le plugin :  
   [https://kevin-benabdelhak.fr/plugins/wp-mp4-to-gif/](https://kevin-benabdelhak.fr/plugins/wp-mp4-to-gif/)

2. Installation sur WordPress :
    1. Rendez-vous dans « Extensions » > « Ajouter ».
    2. Cliquez sur « Téléverser une extension ».
    3. Sélectionnez l’archive ZIP et cliquez sur « Installer ».
    4. Activez le plugin.

3. Utilisation :
    - Accédez à « Médias » > « MP4 → GIF » pour la conversion unitaire.
    - Pour la conversion en masse, sélectionnez vos MP4 dans la Médiathèque (vue liste), puis utilisez l’action de groupe « Convertir en GIF (navigateur) ».
    - Définissez la largeur désirée des GIFs & le nombre d’images/seconde dans l’onglet « Réglages ».

== FAQ ==

= La conversion se fait-elle sur le serveur ? =
Non. Toute la conversion se fait dans votre navigateur, aucun fichier vidéo n’est transmis hors de votre site WordPress.

= Puis-je convertir plusieurs vidéos d’un coup ? =
Oui, via la Médiathèque en mode liste et l’action groupée « Convertir en GIF ».

= Les fichiers GIF générés sont-ils optimisés ? =
Vous pouvez choisir la largeur du GIF et le nombre d’images/seconde dans les réglages pour ajuster la taille finale (plus le GIF est large et fluide, plus il est volumineux).

= Quels sont les prérequis côté navigateur ? =
Fonctionne sur tous les navigateurs modernes (Chrome, Firefox, Edge). La conversion nécessite de laisser la fenêtre ouverte le temps de la génération (temps variable selon la longueur de la vidéo).

= Est-ce compatible multisite WordPress ? =
Oui, chaque site du réseau gère son propre réglage et sa propre bibliothèque.
